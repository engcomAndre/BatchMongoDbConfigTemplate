# Batch Processing with Spring Batch and MongoDB

This project demonstrates a batch processing application using Spring Batch and MongoDB. The application reads items, processes them, and writes the results.

## Technologies Used

- Java
- Spring Boot
- Spring Batch
- MongoDB
- Maven
- Docker

## Prerequisites

- Java 11 or higher
- Maven
- Docker
